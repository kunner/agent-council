## 문서 상태 관리

각 문서 상단에 상태를 명시한다:

```markdown
---
status: TODO | DRAFT | REVIEW | DONE
priority: 1 | 2 | 3 | 4
last_updated: 2026-03-23
---
```

## PLAN.md와의 관계

- **PLAN.md**: 전체 비전, 개요, 로드맵 (경영진/이해관계자용 한 장짜리)
- **상세 문서**: 실제 구현을 위한 엔지니어링 스펙 (개발자용)
- PLAN에서 큰 방향을 잡고, 상세 문서에서 구현 수준으로 풀어낸다
- 상세 문서가 PLAN과 충돌하면 상세 문서가 우선 (더 최신)

## 설정값 단일 출처

`00_설정_참조표.md`는 **모든 설정값의 단일 출처(Single Source of Truth)**다.

포트 번호, 서버 경로, Firestore 컬렉션 경로, tmux 세션 명명 규칙, Firebase 설정, Claude Code 세션 제한, Enum 정의, UI 설정값 등 **구체적인 숫자·문자열 값이 필요한 경우 이 파일을 먼저 확인**한다.

각 상세 문서에서 설정값을 변경해야 할 경우, `00_설정_참조표.md`를 업데이트하고 관련 문서에 전파한다.
